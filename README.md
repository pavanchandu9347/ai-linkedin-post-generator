# AI LinkedIn Post Generator

A web application that generates professional LinkedIn posts using AI.

## Features

- Generate LinkedIn posts from achievements/topics
- Multiple tones (Professional, Casual, Motivational)
- Flask backend
- Responsive frontend
- OpenAI API integration

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenAI API

## Installation

```bash
git clone https://github.com/pavanchandu9347/ai-linkedin-post-generator.git
cd ai-linkedin-post-generator

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

## Run

```bash
cd backend
python3 app.py
```

Open:

```text
http://127.0.0.1:5000
```
